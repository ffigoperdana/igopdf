Kebijakan Privasi Internal

igo digunakan sebagai aplikasi internal. Akses pengguna dilindungi oleh login, CAPTCHA, session cookie HttpOnly, dan role-based access control.

File PDF yang diproses melalui alat browser tidak dikirim ke backend aplikasi untuk diproses. Backend digunakan untuk autentikasi, manajemen sesi, dan administrasi pengguna.

Sistem dapat menyimpan data akun, sesi, percobaan login, dan audit aktivitas admin untuk kebutuhan keamanan operasional.

Hubungi administrator internal bila akun perlu dinonaktifkan, password direset, atau akses diperbarui.
